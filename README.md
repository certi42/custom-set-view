# Custom Set View

A web application for searching and building decks out of cards from custom Magic: the Gathering sets.

## Project setup
```
yarn
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
